# Sizing System Props

요소의 사이즈를 위한 prop입니다.

## Sizing Props

| 속성      | 타입           | 설명             | 크로스플랫폼 지원 |
| :-------- | :------------- | :--------------- | :---------------- |
| width     | number, string | 요소의 너비      | ✅                |
| height    | number, string | 요소의 높이      | ✅                |
| minWidth  | number, string | 요소의 최소 너비 | ✅                |
| minHeight | number, string | 요소의 최소 높이 | ✅                |
| maxWidth  | number, string | 요소의 최대 너비 | ✅                |
| maxHeight | number, string | 요소의 최대 높이 | ✅                |
