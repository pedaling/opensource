# Position System Props

## Position Props

| 속성       | 타입                   | 설명                                           | 크로스플랫폼 지원 |
| :--------- | :--------------------- | :--------------------------------------------- | :---------------- |
| `position` | `absolute`, `relative` | `absolute`시 요소가 부모기준으로 덮어씌웁니다. | ✅                |
| `top`      | `number`               | `absolute`시 요소의 상단 위치를 지정합니다.    | ✅                |
| `left`     | `number`               | `absolute`시 요소의 좌측 위치를 지정합니다.    | ✅                |
| `right`    | `number`               | `absolute`시 요소의 우측 위치를 지정합니다.    | ✅                |
| `bottom`   | `number`               | `absolute`시 요소의 하단 위치를 지정합니다.    | ✅                |
| `zIndex`   | `number`               | 요소의 z-index를 지정합니다.                   | ✅                |
