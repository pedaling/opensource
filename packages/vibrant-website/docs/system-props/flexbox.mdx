# Flexbox System Props

Flex Layout 관련 스타일을 지정할 수 있는 Prop입니다.

## Flexbox Props

| 속성           | 타입                                                           | 설명                                        | 크로스플랫폼 지원 |
| :------------- | :------------------------------------------------------------- | :------------------------------------------ | :---------------- |
| flexGrow       | number                                                         | flex-grow                                   | ✅                |
| flexShrink     | number                                                         | flex-shrink                                 | ✅                |
| flexBasis      | string                                                         | flex-basis                                  | ✅                |
| flexDirection  | `column-reverse`, `column`, `row-reverse`, `row`               | flex-direction                              | ✅                |
| flexWrap       | `nowrap`, `wrap-reverse`, `wrap`                               | flex-wrap                                   | ✅                |
| alignContent   | `stretch`, `center`, `flex-start`, `flex-end`, `space-between` | align-content                               | ✅                |
| alignItems     | `stretch`, `center`, `flex-start`, `flex-end`                  | align-items                                 | ✅                |
| alignSelf      | `auto`,`stretch`, `center`, `flex-start`, `flex-end`           | align-self                                  | ✅                |
| justifyContent | `stretch`, `center`, `flex-start`, `flex-end`, `space-between` | justify-content                             | ✅                |
| justifyItems   | `stretch`, `center`, `flex-start`, `flex-end`                  | justify-items                               | ✅                |
| justifySelf    | `auto`, `stretch`, `center`, `flex-start`, `flex-end`          | justify-self                                | ✅                |
| order          | number                                                         | flex layout 아이템 배치의 순서를 설정합니다 | Web Only          |
| rowGap         | number                                                         | flex layout 아이템의 수직 간격을 설정합니다 | Web Only          |
| columnGap      | number                                                         | flex layout 아이템의 수평 간격을 설정합니다 | Web Only          |
