# Input System Props

Text Input 관련 스타일을 지정할 수 있는 prop입니다.

## Input Props

| 속성                | 타입                      | 설명                   | 크로스플랫폼 지원 |
| :------------------ | :------------------------ | :--------------------- | :---------------- |
| placeholderColor    | `ColorToken`              | placeholder 색상       | O                 |
| hideInputSpinButton | `boolean`                 | input spin button 숨김 | Web Only          |
| textAlign           | `center`, `left`, `right` | input text 정렬        | O                 |
