# Typography System Props

폰트의 스타일을 설정하는 prop입니다.

## Text Props

| 속성       | 타입                          | 설명                          | 크로스플랫폼 지원 |
| :--------- | :---------------------------- | :---------------------------- | :---------------- |
| typography | TypographyKindToken           | 텍스트의 스타일을 설정합니다. | ✅                |
| fontFamily | string                        | 폰트 이름                     | ✅                |
| fontSize   | number                        | 폰트 크기                     | ✅                |
| fontWeight | TypographyWeightToken         | 폰트 두께                     | ✅                |
| fontStyle  | `italic`, `normal`, `oblique` | 폰트 스타일                   | ✅                |
| lineHeight | number                        | 줄 간격                       | ✅                |
