# Color System Props

색상을 설정할 수 Prop입니다.

주로 텍스트 및 텍스트 입력창의 색을 설정하기 위해 사용합니다.

배경색에 대해서는 [Background System Prop](background)를 참고해주세요.

## Color Props

| 속성  | 타입         | 설명                      | 크로스플랫폼 지원 |
| :---- | :----------- | :------------------------ | :---------------- |
| color | `ColorToken` | 텍스트 색상을 설정합니다. | ✅                |
