# Text System Props

텍스트의 스타일을 설정하는 prop입니다.

## Text Props

| 속성          | 타입                                                    | 설명                              | 크로스플랫폼 지원 |
| :------------ | :------------------------------------------------------ | :-------------------------------- | :---------------- |
| letterSpacing | number                                                  | 글자 간격을 설정합니다.           | ✅                |
| textAlign     | `center`, `left`, `right`                               | 텍스트의 정렬을 설정합니다.       | ✅                |
| textTransform | `capitalize`, `lowercase`, `uppercase`, `none`          | 텍스트의 대소문자를 설정합니다.   | ✅                |
| whiteSpace    | `normal`, `pre-line`, `pre-wrap`, `break-spaces`, `pre` | 텍스트의 공백을 설정합니다.       | Web Only          |
| wordBreak     | `break-all`, `break-word`. `keep-all`, `normal`         | 텍스트의 줄바꿈을 설정합니다.     | Web Only          |
| wordWrap      | `break-word`. `normal`                                  | 텍스트의 줄바꿈을 설정합니다.     | Web Only          |
| overflowWrap  | `anywhere`, `break-word`, `normal`                      | 텍스트의 줄바꿈을 설정합니다.     | ✅                |
| lineLimit     | number                                                  | 텍스트의 최대 줄 수를 설정합니다. | ✅                |
