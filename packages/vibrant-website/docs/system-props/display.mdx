# Display System Props

## Display Props

| 속성       | 타입                           | 설명                                                                         | 크로스플랫폼 지원 |
| :--------- | :----------------------------- | :--------------------------------------------------------------------------- | :---------------- |
| display    | `flex`, `inline-flex`, `none`  | Layout 종류를 설정하는 속성입니다, `none`시 영역을 차지하지 않습니다.        | ✅                |
| visibility | `visible`, `inherit`, `hidden` | 노출되는 여부를 설정하는 속성입니다. `hidden`이여도 화면의 공간은 차지합니다 | ✅                |
| opacity    | `OpacityToken`, number         | 투명도를 설정하는 속성입니다.                                                | ✅                |
| hidden     | boolean                        | `true`시 `display: none`으로 설정합니다.                                     | ✅                |
