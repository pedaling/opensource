# Transform System Props

요소의 모양을 변환하는 Prop입니다. (주로 애니메이션에서 사용합니다)

## Transform Props

| 속성      | 타입                    | 설명                    | 크로스플랫폼 지원 |
| :-------- | :---------------------- | :---------------------- | :---------------- |
| transform | [Transform](#transform) | 변환 효과를 적용합니다. | ✅                |

## 타입

### Transform

| 값         | 설명                |
| :--------- | :------------------ |
| rotate     | 회전                |
| rotateX    | X축을 기준으로 회전 |
| rotateY    | Y축을 기준으로 회전 |
| rotateZ    | Z축을 기준으로 회전 |
| scale      | 크기 조절           |
| scaleX     | X축으로 크기 조절   |
| scaleY     | Y축으로 크기 조절   |
| translateX | X축으로 이동        |
| translateY | Y축으로 이동        |
| skewX      | X축으로 기울임      |
| skewY      | Y축으로 기울임      |
