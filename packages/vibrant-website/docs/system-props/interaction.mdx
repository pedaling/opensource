# Interaction System Props

## Interaction Props

| 속성            | 타입                         | 설명                                                      | 크로스플랫폼 지원 |
| :-------------- | :--------------------------- | :-------------------------------------------------------- | :---------------- |
| `cursor`        | `default`, `pointer`, `text` | 요소에 마우스를 올릴 때 표시되는 커서의 모양을 지정합니다 | Web Only          |
| `pointerEvents` | `auto`, `none`               | 요소의 포인터 이벤트를 활성화 또는 비활성화합니다         | ✅                |
