# Icons

`<Icon />` 은 Vibrant UI에서 사용되는 아이콘 컴포넌트입니다. 아래에는 사용 가능한 아이콘 목록이 있고, 클릭 시 아이콘 이름이 복사됩니다.
 
<IconMaps />

## How to use

```jsx
import { Icon } from '@vibrant-ui/icons';

<Icon.Add.Fill size={24} />
<Icon.Add.Regular size={24} />
<Icon.Add.Thin size={24} />
```
