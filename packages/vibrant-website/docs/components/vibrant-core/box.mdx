# Box

Box는 [시스템 속성](/docs/system-props/introduce)으로 정의된 다양한 스타일 속성으로 네이티브, 웹에서 동일하게 스타일이 적용된 UI를 쉽게 구현할 수 있도록 하는 스타일 유틸리티 컴포넌트입니다.

```jsx live
<Box background="primary" height={100} width={100} />
```

## How to use

```jsx
import { Box } from '@vibraBoxnt-ui/core';
```
