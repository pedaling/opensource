# TableFooter

```jsx live
<TableFooter
  total={101}
  pagination={true}
  pageSize={5}
  pageCount={10}
  pageSizeOptions={[5, 10]}
/>
```

## How to use

```tsx
import { TableFooter } from '@vibrant-ui/components';
```

## Properties

### TableFooter

| Prop             | Type                         | Default | Description                                                           |
| ---------------- | ---------------------------- | ------- | --------------------------------------------------------------------- |
| total            | `number`                     |         | 데이터의 개수를 지정합니다.                                           |
| showTotal        | `number`                     |         | 데이터의 개수의 렌더 여부를 결정합니다.                               |
| pagination       | `boolean`                    |         | 페이지네이션의 렌더 여부를 결정합니다.                                |
| defaultPageSize  | `number`                     |         | 테이블의 페이지 당 열의 개수를 지정합니다.                            |
| currentPage      | `number`                     |         | 페이지네이션의 현재 페이지를 지정합니다.                              |
| onPageChange     | `(page: number) => void`     |         | 테이블 푸터의 페이지네이션 버튼이 눌렸을 때 실행할 동작을 지정합니다. |
| pageSizeOptions  | `number[]`                   |         | 선택가능한 페이지 당 열의 개수 옵션을 지정합니다.                     |
| onPageSizeChange | `(pageSize: number) => void` |         | 페이지 당 열의 개수가 바뀔 때 실행할 동작을 지정합니다.               |
