# Spinner

`Spinner` 컴포넌트는 로딩 상태를 표시하는데 사용됩니다.

```jsx live
<Spinner size="lg" />
```

`Spinner`의 컬러는 `onColor`로 배경색에 따라 자동으로 결정됩니다.

## How to use

```jsx
import { Spinner } from '@vibrant-ui/components';
```

## Properties

| Prop     | Type                                 | Default | Description                    |
| -------- | ------------------------------------ | ------- | ------------------------------ |
| size(\*) | `xs` \| `sm` \| `md` \| `lg` \| `xl` | -       | `Spinner`의 크기를 설정합니다. |
